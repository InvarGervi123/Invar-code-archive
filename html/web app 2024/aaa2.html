תשובה שאלה
פרוטוקול HTTP הוא פרוטוקול המשמש להעברת מידע בין רשתות המחשבים באינטרנט. מהו פרוטוקול HTTP?
פרוטוקול FTP הוא פרוטוקול המשמש להעברת קבצים בין מחשבים ברשת האינטרנט. מהו פרוטוקול FTP?
כתובת IP היא כתובת מספרית המשמשת לזיהוי ולאיתור של מחשב או מכשיר ברשת האינטרנט. מהי כתובת IP?
פרוטוקול DNS הוא פרוטוקול המשמש לתרגום שמות התחומים לכתובות IP. מהו פרוטוקול DNS?
שמו של הפרוטוקול המתרגם שמות תחומים לכתובת IP הוא DNS (Domain Name System). מהו שמו של הפרוטוקול המתרגם שמות תחומים לכתובת IP?
היישום המתרגם קוד אתר ומציג אותו למשתמש נקרא דפדפן אינטרנט. מהו שמו של היישום המתרגם את קוד האתר ומציג אותו למשתמש?
